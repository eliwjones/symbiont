Requirements
============

* python3

Quick Start
===========

1. Clone repo and run code:
```
$ git clone git@github.com:eliwjones/symbiont.git
$ cd symbiont
$ ./random
```

2. Press random keys on keyboard.  When the keypress buffer is full enough, it will extract random data to flush to stdout.
